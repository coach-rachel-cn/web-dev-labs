Understanding the CSS Position Property

This demo illustrates how the values for the CSS position property affect HTML element placement on a web page.

1. Static (Default)

Static Box (Default)

This box has position: static; This is the default position value. It flows normally in the document and cannot be moved with `top`, `bottom`, `left`, `right` properties.

2. Relative

Relative Box

This box has position: relative;. It stays in the document flow but can be moved relative to its normal position using `top`, `bottom`, `left`, or `right`. It can also serve as the positioning context for `absolute` children.

3. Absolute

This container relative position to act as the positioning context for the absolute box inside.

Absolute Box

This box has position: absolute;. It's removed from the document flow and positioned relative to its nearest *positioned* ancestor (like its `.absolute-container` parent). If no positioned ancestor exists, it's positioned relative to the ``.

4. Fixed

Fixed Box

Scroll down to see this box stay in place! A position: fixed; box is positioned relative to the browser viewport. It stays in the same spot even when the page scrolls. Use `top`, `bottom`, `left`, `right` to place it.
